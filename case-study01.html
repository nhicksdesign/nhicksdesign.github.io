personal identity

After getting more and more exposure to the UI/UX and Web Design world, I decided it was time to overhaul my personal identity and clean up my portfolio site. I knew I wanted to build the site from scratch, as opposed to my previous site that I built on Bootstrap. I also knew that I wanted to make it more minimal and type focused than my last site. With that in mind, I updated my logo as well.

I created a much more minimal version of my logo first. Then laid out a super clean type focused site. It needed to be lightweight to reduce the need of loaders and I didn't want it to be too over the top with jQuery effects.

Graphic Design

Along with revamping my personal logo, I pulled a lot of the imagery out of my identity and refocused on clean minimal Typography. By balancing color, layout, and white space, I was able to create a sleek new look.

Web Design

Utilizing HTML5, CSS3, and jQuery, I was able to build a much faster and cleaner framework for this new site. By getting rid of Bootstrap and instead utilizing a lightweight CSS Grid, I was able to make a tidy site with a fast load time.

UI/UX Design

Leveraging UI eye tracking studies, UI best practices, and clean material design, I tried to make sure this site was intuitive and easy to use over everything else.
