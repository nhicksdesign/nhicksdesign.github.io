upstate elevator

Upstate Elevator needed an ecommerce site ASAP to keep up with competitors and wanted to use the Shopify platform. I helped devise a clean and charming web identity and laid the design out within it, added custom code injection where needed, and helped them get their online presence off the ground.

Utilizing their logo and packaging as a jump off point, I refined clean and regal color sets, font systems, and product imagery to really make their site stand out.

Web Design

Although this site was built using Shopify, I was able to leverage web colors, typography, and some subtle code work to really make this site shine. Shopify can be limiting, but I really wanted to make sure this site stood out among all other sites made using this platform.
