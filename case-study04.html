IVFCloud

Business Recovery Center needed an informational site created for their new cloud platform. The project needed to be ready in around a week, so I leaned heavily on Materialize to get this done in time. I was able to finish with enough time to add in some custom animations and make things a bit more succinct.

After creating a logo for the product, I coded a site that reflected a lot of the same design ideas. I had helped the client with some presentation designs as well, so I decided to incorporate some of the imagery from that to keep the whole experience consistant.

Graphic Design

I created a logo set for this product, as well as generating some designs and lay outs for the presentations that were delivered to the customer.

Web Design

Utilizing HTML5, CSS3, and jQuery, I built a clean site that worked with a balance of image, color, and typography. It was a very tight deadline, so I leveraged Materialize's incredible CSS libraries to make styling a bit quicker.

UI/UX Design

I utilized a lot of the same schools of thought for the interface of this site. Along with making it responsive and mobile friendly, I wanted it to work as seamlessly as possible. I included a page loader to keep the user's attention while the high res photos loaded. The site was also made using all UI friendly color hexes.
